This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Overview

This is a static academic personal website for Weijun Wang (王蔚峻), a Postdoctoral Fellow at Tsinghua University. It was originally generated by [jemdoc](http://jemdoc.jaboc.net/) but is now maintained as plain HTML.

There is no build system, package manager, or framework. The site is deployed directly as static files (GitHub Pages).

## Local Development

Open any `.html` file directly in a browser, or serve locally:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000`.

## Architecture

- All pages are standalone `.html` files — each one manually duplicates the left sidebar navigation. When adding a new page or changing navigation items, **every HTML file must be updated**.
- Styling comes from `jemdoc.css` (base jemdoc layout) and `theme.css` (custom overrides).
- `theme-switcher.js` applies a day/night CSS class to `document.body` based on the hour.
- `mysite.conf` is the original jemdoc configuration (Google Analytics, footer template). It is not used at runtime but kept as reference.
- `ref/` contains linked PDF papers. `share/` contains certificates and shared papers. `img/` contains photos and icons.

## Content Conventions

- Publication entries in `papers.html` use `<span>[VENUE]</span>` for venue labels (styled purple: `rgb(116, 52, 133)`).
- `<u>underscore</u>` wraps student/mentee names in author lists.
- The "Research Interests" section on `index.html` groups papers under topic headings with inline links; older topic groups are wrapped in `<div style="opacity: 0.3;">` to visually fade them.
- News items go inside the `.scrollable` div in `index.html`, newest first.
